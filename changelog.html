Version History

0.4.1

Released on 3 March 2010

No code changes in release. Migrated source repository from Google Code to Github, will move issues and wiki etc over int he future
Features
  • Major overhaul of build script and procedure.
  • Updated OSX SWT version to 3.6M3 so Drag and Drop now works

0.4

Released on 10 Nov 2009

Features
  • Updated to use TVRage's new xml feed format, the program should work now
  • Drag and drop of files is now supported
  • You can now manually override the show's title. In addition, pressing enter while inside the show title or format fields will apply the changes
  • Removed success dialog, now prints success message in the status bar (bottom of window)

0.3

Released on 22 April 2009

Features
  • Updated to use TVRage's new xml feed format, the program should work now
  • Drag and drop of files is now supported
  • You can now manually override the show's title. In addition, pressing enter while inside the show title or format fields will apply the changes
  • Removed success dialog, now prints success message in the status bar (bottom of window)
Bug Fixes
  • Fixed crashing on Mac OS X

0.2

Released on 26 April 2008

Features
  • Show and Episode information is now provided by TVRage's xml feeds, http://www.tvrage.com/xmlfeeds.php
  • Users can now provide a format string to rename the files by
  • Users are now provided with list of options for the show, in case the selected show is incorrect
  • Now correctly handles nested folders, e.g.D:\tv\Battlestar Galactica\*Season Four*\battlestar.galactica.402.mkv, Previously only single level folders were supported, e.g. D:\tv\Battlestar Galactica\battlestar.galactica.402.mkv
